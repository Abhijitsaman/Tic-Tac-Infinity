# Tic Tac Infinity 🎮

A premium, production-ready Tic Tac Toe game with AI, tournaments, achievements, and more.

## Features

- 🎯 **Multiple Game Modes**: Classic, AI, Tournament, Time Attack
- 🤖 **Smart AI**: From Easy to Impossible (Minimax algorithm)
- 🏆 **Achievement System**: 9 achievements with XP rewards
- 📊 **Statistics Dashboard**: Beautiful charts and analytics
- 🎨 **5 Themes**: Cosmic, Neon Purple, Lava Red, Ocean Blue, Emerald Green
- 🔊 **Audio System**: Background music and sound effects
- 🔄 **Replay System**: Watch your past matches move-by-move
- 💾 **Local Storage**: All data persisted locally
- 📱 **Responsive**: Works on all devices

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- Recharts
- Lucide React
- React Confetti
